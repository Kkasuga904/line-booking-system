📅 予約管理モーダル - 使用例

説明： このページは既存のウェブサイトにモーダル形式で予約管理機能を追加する例です。

バックエンドAPI: https://line-booking-api-dxp5vd3wbq-an.a.run.app

予約一覧

「予約一覧を更新」ボタンをクリックしてください

💡 既存サイトへの組み込み方法

<!-- 1. JavaScriptファイルを読み込む -->
<script src="modal-booking.js"></script>

<!-- 2. 予約作成ボタンを追加 -->
<button onclick="openBookingModal()">
    新規予約
</button>

<!-- 3. 予約一覧を表示（オプション） -->
<div id="myReservations"></div>
<script>
    // 予約一覧を表示
    displayReservations('myReservations');
</script>

🔧 APIの使用方法

// 予約一覧を取得
const reservations = await BookingAPI.loadReservations();

// 予約を作成
const result = await BookingAPI.createReservation({
    date: '2025-01-10',
    time: '12:00',
    customer_name: '山田太郎',
    phone: '090-1234-5678',
    people: 2,
    notes: 'アレルギー：なし',
    status: 'confirmed'
});

// 予約を削除
await BookingAPI.deleteReservation(reservationId);
